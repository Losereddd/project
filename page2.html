The main character - a man (by default his name is Player 1, but the user can choose any name) - receives the MiSide application on his smartphone, sent by a certain M.K. In it you need to look after a female character named Mita. The app itself is a 2D life simulation/mobile Tamagotchi game with a top-down view.
MiSide is an adventure computer game with horror elements in anime style from indie developers AIHASTO. The beta version of the game was released on August 18, 2023[1], and the official release took place on December 11, 2024[2]. According to the plot of MiSide, the main character moves inside a mobile game about a girl named Mita[3].
